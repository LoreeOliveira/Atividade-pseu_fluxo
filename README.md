# Atividade 18 - Lógica de Programação 💻

Este repositório contém os exercícios práticos da **Atividade 18** desenvolvidos durante o curso de **Desenvolvimento de Sistemas no SENAI**. O foco desta atividade foi o fortalecimento da lógica de programação através da criação de algoritmos e representações visuais.

## 🚀 Sobre o Projeto

O objetivo foi resolver problemas propostos utilizando duas abordagens distintas para cada desafio:
1.  **Pseudocódigo:** Escrito em Portugol através do **VisualG**.
2.  **Fluxograma:** Representação gráfica da lógica através do **Flowgorithm**.

## 🛠️ Tecnologias e Ferramentas

* [VisualG](http://visualg3.com.br/) (Interpretador de algoritmos)
* [Flowgorithm](http://www.flowgorithm.org/) (Criação de fluxogramas)
* Git/GitHub (Versionamento de código)

## 📂 Exercícios Resolvidos

Abaixo estão listados os desafios implementados neste repositório:

* **Idade:** Verifica categorias ou permissões baseadas na idade informada.
* **Média:** Calcula a média aritmética de notas e define o status de aprovação.
* **Senha:** Sistema de validação de acesso com senha.
* **Soma:** Algoritmo simples para somar valores numéricos.
* **Voto:** Verifica a obrigatoriedade ou possibilidade de voto com base na idade.
* **Comissão de Vendedor:** Calcula o valor a receber sobre vendas realizadas.
* **Volume de Paralelepípedo:** Cálculo matemático de volume (comprimento × largura × altura).

## 📁 Estrutura de Arquivos

* `.alg`: Arquivos de código-fonte para abertura no **VisualG**.
* `.fprg`: Arquivos de fluxo para abertura no **Flowgorithm**.
